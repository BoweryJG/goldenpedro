Dashboard Routes:

Main Dashboard Login Page Direct Access SMS Queue

Navigation Structure:

  • Protected Routes:
    • /dr-pedro/dashboard - Main dashboard (requires authentication)
    • /dr-pedro/sms - SMS Queue (requires authentication)
  • Public Routes:
    • /dr-pedro/login - Login page
    • /dr-pedro/direct-access - Direct access bypass
    • /dr-pedro/unauthorized - Unauthorized access page

Dashboard Features:

  • Watch Dashboard - Analytics gauges
  • Daily Schedule
  • Week Overview
  • Staff Schedule
  • Analytics
  • Voice Calls
  • Voice AI Settings
  • Phone Numbers
  • Subdomains (TMJ, Implants, Robotic, MedSpa)

Navigation Implementation:

The dashboard uses React Router for navigation. All dashboard routes are protected by authentication.

Navigation is handled through:

  • React Router's useNavigate hook
  • onClick handlers that call navigate()
  • Protected routes that check authentication status
